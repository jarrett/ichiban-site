  • Ichiban automatically compiles your site whenever you save a source file. So there's no need to switch to the terminal and run a compiler every time you change something. To use automatic compilation, start the watcher:

    cd my-project
    ichiban watch

    Now change a file, save it, and refresh your browser. Your changes will be reflected immediately.

  • Sometimes you need to generate HTML from a data source, like a CSV or XML file. Ichiban has you covered. You can write a loop inside a single page, or even generate a completely new page for each loop iteration.

    Say you have a bunch of employees in a CSV file. You can write a script like this:

    CSV.parse('employees.csv').each do |first, last|
    generate '_employee.html', 'employees/' + last, first: first, last: last
    end

    The above will use the template _employee.html to generate an individual page for each employee in the CSV file.

  • Ichiban is Ruby through-and-through. With ERB templates, you can embed any logic you can imagine into your HTML. Write custom helper methods, use conditionals, generate sophisticated nav menus, loop over data—it's all possible with Ichiban.

  • Ichiban has plenty of handy features. Some standouts:

    • Reusable layouts, and even nested layouts
    • Nav menu generator
    • SCSS
    • Markdown
    • JS minification (with source maps for debugging)
    • EJS templates
