Basic HTML Pages

To add a basic HTML page to your site, create it in the html folder. For example:

<%= code_block(4, %Q( # In one terminal: cd my-site ichiban watch # In another terminal: cd my-site edit html/contact-us.html )) %>

When you save changes to contact-us.html, the watcher automatically compiles the page, placing the result in the compiled folder.
