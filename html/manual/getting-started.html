Getting Started

Install Ichiban and create a new project:

<%= code_block(4, %Q( gem install ichiban ichiban new my-site cd my-site bundle install )) %>

If you're using Apache, set up a virtual host:

<%= code_block(4, %Q( Listen 3000 DocumentRoot "/Users/username/Sites/my-site/compiled" AllowOverride All Order allow,deny Allow from all )) %>

Replace the paths in the example above with the right ones for your folder layout. If you're not using Apache, start your webserver of choice and set its document root to the compiled folder.

To make sure your virtual host is working, point your browser to http://localhost:3000/.

Start the watcher.

<%= code_block(4, %Q( ichiban watch # Or, to avoid possible gem version issues: bundle exec ichiban watch )) %>

With this running, any changes you make will be reflected in the compiled folder automatically. Open html/index.html in your text editor, make a change, and save it. Refresh the page in your browser. You should see your change.
